# Python for AI - Complete Beginner Course

## High-Level Overview
This repository tracks my progress and serves as a comprehensive codebase as I follow along with the "Python for AI - Complete Beginner Course" by Dave Ebbelaar from Datalumina. The purpose of this repo is to capture practical, battle-tested Python code specifically tailored for artificial intelligence development. Whether you have no prior programming experience or are switching from another language, this repository documents the journey from writing your first lines of code to building real AI applications.

## How We Are Doing It
This project is built by coding along with the [Python for AI Beginner Course on YouTube](https://youtu.be/ygXn5nV5qFc) and referencing the [companion documentation handbook](https://python.datalumina.com/llms.txt). 

Instead of jumping straight into isolated code snippets, we are establishing a professional development environment first. This includes using tools like Visual Studio Code, setting up proper Git version control, and using virtual environments and modern package managers like `uv` to cleanly manage dependencies. Everything is practiced with hands-on, real-world examples.

## The Venture Goal
The ultimate goal of this venture is to master the Python fundamentals required to confidently build custom AI applications, work with data, and create AI agents (like your own AI chat assistant that can interact with your data). By the end of this project, this repository will serve as a complete toolkit for developing AI solutions and interacting with external APIs securely and efficiently.

## What This Course Covers (High-Level)
This project is broken down into several core milestones:

*   **Setup & Professional Workflow:** Installing Python, configuring VS Code, managing packages/dependencies with `pip` and `uv`, and establishing isolated virtual environments.
*   **Python Basics:** Mastering foundational concepts like variables, data types (strings, integers, booleans), control flows (if/else statements, loops), and data structures (lists, dictionaries, sets).
*   **Building Programs:** Writing reusable functions, handling errors gracefully with `try/except` blocks, and an introduction to Object-Oriented Programming (Classes).
*   **External Tools & APIs:** Importing external libraries to scrape data, interacting with APIs using the `requests` library, analyzing data with `pandas`, and plotting charts with `matplotlib`.
*   **Production-Ready Practices:** Organizing code across multiple modules, formatting code with `Ruff`, hiding sensitive API keys in `.env` files, and using Git/GitHub for version control.
